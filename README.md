# RUBYメソッド辞典
Ruby、Railsメソッドをメモするためのアプリです。  

## 開発言語
- Ruby 3.1.3
- Rails 6.1.7

## データーベース
- PostgreSQL

## 実行手順
1. git@github.com:cclemon3248/ruby_method_app.git  
1. cd ruby_method_app  
1. bundle install  
1. rails db:create  
1. rails db:migrate  
1. rails db:seed
1. rails s
